Privacy Policy

Steam Linker temporarily stores data related to messages that had the "Get Direct Link" button pressed. This includes the following:


  • User ID The user that used the button
  • User Locale
  • Message ID
  • Message Author ID
  • Message Content Message content is not saved, Only used to extract links containing: "steamcommunity.com" or "store.steampowered.com"
  • Server ID
  • Channel ID
